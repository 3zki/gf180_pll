# gf180_pll
Analog PLL testbench

# Files
* pll_bench - PLL and its benchmark
* pfd - PFD 
  * dff_pfd - Special DFF for PFD
* cp - Charge pump
* lf - Loop filter
* ctrlsel - Vctrl selector
* vco - VCO
  * inv_bias - Special inverter for the ring oscillator
* fdiv - Clock divider by 2
* sw - Output switch
* nand2 - 2 input NAND
* nand3 - 3 input NAND
